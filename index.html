How Foresight Studio Builds Personas from CDP Data

Foresight Studio transforms raw AEP segment exports into psychologically rich synthetic personas. Here is exactly what happens under the hood.

The Translation Pipeline

Your CSV contains 35+ columns of aggregate segment data — demographics, behavioral metrics, satisfaction scores, campaign response rates, and CRM tags. Foresight Studio does not simply list this data back. It interprets correlations between data points to generate psychological insights. For example: a 28% email open rate + 58% OTA booking rate + 0.62 churn probability tells the engine this persona is price-driven, brand-agnostic, and only responds to specific dollar-off messaging — not percentage discounts.

Data Sources from AEP

Segmentation Service defines which groups to build personas for. Profile Access API provides identity, loyalty, and preference attributes (aggregated per segment). Query Service computes behavioral metrics from Experience Events — email engagement, booking patterns, app sessions, campaign response. All three combine into one CSV row per segment.

Cloud Storage Integration

AEP Destinations can export to S3, Azure Blob, or GCS on a schedule. Query Service runs daily aggregation, writes to a derived dataset, and that dataset auto-exports to cloud storage. Foresight Studio can fetch directly from your bucket via pre-signed URL — no manual copy-paste needed.

How the Research Agent Works

Foresight Studio's Research Agent receives the full experiment context — all persona profiles, scores, reasoning, and creative details — with every question. It traces causal chains across four layers of evidence to deliver specific, actionable insights.

Four-Layer Evidence Chain

When you ask "Why did Near-Retirees score low on Trust?", the agent cross-references: (1) The persona profile — "distrusts flashy marketing, wants credentials". (2) The Trust score — 3/10. (3) The evaluation reasoning — "no specific numbers, feels evasive". (4) Your actual creative copy — what triggered that reaction. It then synthesizes a specific recommendation grounded in all four layers.

Conversation Memory

The agent maintains full chat history within a session. You can explore a creative direction across multiple turns: "Rewrite the headline" → "Now adapt it for another segment" → "Make it more urgent" — each turn builds on previous context.

Smart Suggestions

Suggestion buttons are computed from your actual results — they identify the lowest-scoring persona, weakest dimension, and strongest performer, generating targeted questions that address the biggest opportunities first.
